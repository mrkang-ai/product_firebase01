# Blueprint: 온라인 도구모음

## 1. 프로젝트 개요

일상생활에 유용한 간단한 웹 도구들과 게임을 제공하는 것을 목표로 하는 웹사이트입니다. 복잡한 설치 과정 없이 누구나 웹 브라우저만으로 쉽게 사용할 수 있는 편의 기능들을 제공합니다. 

본 프로젝트는 Google AdSense 광고 모델을 염두에 두고 있으며, 이를 위해 각 도구 페이지는 단순한 기능 제공을 넘어 관련 정보를 포함한 풍부한 콘텐츠를 제공하도록 구성되었습니다.

## 3. 기능 명세

### 3.1. 메인 페이지 (`/`)

- **목적:** 사이트의 첫인상을 결정하고, 제공하는 도구들을 소개하는 포털 역할을 합니다.
- **주요 기능:**
    - 사이트의 정체성과 목적을 설명하는 소개 문구.
    - 각 도구 페이지로 연결되는 내비게이션 링크.
    - 사이트의 미래 확장 가능성에 대한 안내.

### 3.2. 로또 번호 생성기 (`/lotto/`)

- **목적:** 사용자에게 행운의 로또 번호를 생성해주고, 로또와 관련된 유용한 정보를 제공하여 흥미를 유발하고 페이지 체류 시간을 늘립니다.
- **주요 기능:**
    - **동행복권 바로가기:** 공식 사이트로 연결되는 외부 링크 버튼.
    - **최신 당첨 정보 (예시):** 최신 회차의 당첨 번호와 1등 당첨금을 예시로 보여주어 사용자의 흥미를 유도합니다.
    - **번호 생성기:** '번호 생성하기' 버튼 클릭 시, 1~45 사이의 중복되지 않는 6개 숫자를 생성하여 화면 하단에 목록 형태로 계속 추가합니다.
    - **로또 정보:** 로또의 역사, 역대 최고/최저 당첨금 정보를 제공하여 콘텐츠의 깊이를 더합니다.
    - **광고 슬롯:** Google AdSense 광고가 삽입될 영역을 명시적으로 표시합니다.

### 3.3. 글자 수 세기 (`/TextCount/`)

- **목적:** 보고서, 자기소개서, SNS 게시물 등 텍스트 길이에 제약이 있는 글을 작성할 때 사용자에게 실시간으로 글자 수 정보를 제공하여 편의를 돕습니다.
- **주요 기능:**
    - **실시간 텍스트 분석:** 사용자가 텍스트 입력창에 내용을 입력하는 즉시 아래 항목들을 계산하여 보여줍니다.
        - 공백 포함 글자 수
        - 공백 제외 글자 수
        - 단어 수
        - 줄 수
    - **초기화:** '초기화' 버튼으로 입력된 텍스트와 모든 카운트를 한 번에 지울 수 있습니다.
    - **도구 안내:** 도구의 활용 방안에 대한 간단한 설명을 제공합니다.
    - **광고 슬롯:** Google AdSense 광고가 삽입될 영역을 명시적으로 표시합니다.

### 3.4. 오늘 뭐 먹지? (`/eat/`)

- **목적:** 매일 점심, 저녁 메뉴 선택에 어려움을 겪는 사용자에게 간단한 버튼 클릭만으로 랜덤하게 음식을 추천해주는 도구입니다.
- **주요 기능:**
    - **메뉴 추천:** '메뉴 추천해 줘!' 버튼을 누르면 3초 동안 메뉴를 탐색하는 애니메이션이 표시된 후, 랜덤으로 선택된 음식 메뉴와 이미지를 보여줍니다.
    - **메뉴 탐색 애니메이션:** 메뉴를 선택하는 동안 여러 음식 이미지가 빠르게 전환되며 시각적인 재미를 더합니다.
    - **음식 정보 제공:** 추천된 음식에 대한 간단한 설명을 함께 제공하여 사용자의 이해를 돕습니다.
- **버그 수정:**
    - **이미지 경로 오류 수정:** 메뉴 탐색 중 및 결과 표시 화면에서 이미지 경로가 잘못 설정되어 이미지가 표시되지 않던 문제를 수정했습니다. `main.js` 파일의 `getImageUrl` 함수에서 `eat/` 접두사를 제거하여 올바른 이미지 경로(`images/...`)를 사용하도록 변경했습니다.

### 3.5. Dodger 게임 (`/Dodger/`)

- **목적:** 하늘에서 떨어지는 장애물을 피하는 간단한 웹 게임을 제공하여 사용자에게 즐거움을 선사합니다.
- **주요 기능:**
    - **메인으로 이동:** 게임 페이지 상단에 메인 화면으로 돌아갈 수 있는 링크를 추가하여 사이트 내 다른 도구로의 이동을 용이하게 했습니다.
    - **플레이어 조작:** 사용자는 키보드 방향키(←, →)를 사용하여 파란색 사각형 플레이어를 좌우로 움직일 수 있습니다.
    - **장애물 낙하:** 화면 상단에서 빨간색 사각형 장애물이 무작위 위치에서 나타나 아래로 떨어집니다.
    - **진행 시간 표시:** 게임이 시작된 후 경과 시간이 화면 좌측 상단에 실시간으로 표시됩니다.
    - **충돌 감지:** 플레이어가 장애물과 부딪히면 게임이 종료됩니다.
    - **게임 오버:** 게임 오버 시, 최종 진행 시간과 함께 "Game Over!" 알림창이 표시되고 페이지가 새로고침되어 즉시 다시 시작할 수 있습니다.


## 4. 디자인 및 스타일 가이드

- **색상:** 신뢰감을 주는 파란색 계열 (`#004a7c`, `#007bff`)을 주 색상으로 사용하고, 녹색 (`#28a745`)과 노란색 (`#f9a825`)을 포인트 색상으로 활용하여 활기찬 느낌을 더합니다.
- **레이아웃:** 헤더, 메인 콘텐츠, 푸터의 3단 구조를 기본으로 하며, 중앙 정렬된 컨테이너를 사용하여 가독성을 높입니다.
- **폰트:** 가독성이 높은 'Segoe UI' 또는 기본 sans-serif 폰트를 사용합니다.

## 5. 향후 확장 계획

- 달력/날짜 계산기, 학점 계산기, 타이머 등 사용자의 수요가 높은 생활 밀착형 도구들을 추가할 예정입니다.
- 각 도구 페이지의 정보 콘텐츠를 지속적으로 보강하여 검색 엔진 최적화(SEO)를 강화합니다.
- Dodger 게임에 점수 시스템, 난이도 조절, 아이템 등 다양한 기능을 추가하여 재미를 더할 예정입니다.
