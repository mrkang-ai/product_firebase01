# Blueprint: 온라인 도구모음

## 1. 프로젝트 개요

일상생활에 유용한 간단한 웹 도구들과 게임을 제공하는 것을 목표로 하는 웹사이트입니다. 복잡한 설치 과정 없이 누구나 웹 브라우저만으로 쉽게 사용할 수 있는 편의 기능들을 제공합니다.

## 2. 디자인 및 스타일 가이드 (v2 - Blue Edition)

- **프레임워크:** Tailwind CSS를 사용하여 유틸리티-우선 접근 방식으로 전체 사이트의 스타일을 재구성했습니다. 이를 통해 일관되고 유지보수 가능한 디자인 시스템을 구축합니다.
- **색상 팔레트:**
    - **Primary:** `#0ea5e9` (Sky Blue) - 주요 버튼 및 하이라이트
    - **Background (Light):** `#f0f9ff` - 라이트 모드 배경
    - **Background (Dark):** `#0f172a` - 다크 모드 배경
    - **Text (Main):** `#0f172a` - 기본 텍스트
    - **Text (Muted):** `#64748b` - 보조 텍스트
    - **Card (Light):** `#ffffff` - 라이트 모드 카드 배경
    - **Card (Dark):** `#1e293b` - 다크 모드 카드 배경
- **폰트:** 전반적인 가독성과 현대적인 느낌을 위해 'Plus Jakarta Sans'를 기본 폰트로 사용합니다.
- **아이콘:** 'Material Symbols'를 사용하여 일관되고 깔끔한 아이콘 시스템을 적용합니다.
- **레이아웃:**
    - **헤더:** `backdrop-blur` 효과가 적용된 반투명 배경의 고정 헤더. 로고, 네비게이션, 언어 및 공유 버튼이 포함됩니다.
    - **카드 디자인:** 서비스 목록, 정보 섹션 등 대부분의 콘텐츠는 그림자 효과와 둥근 모서리가 적용된 카드 형태로 제공되어 입체감과 구분감을 줍니다.
    - **반응형:** 모바일, 태블릿, 데스크탑 등 모든 화면 크기에서 최적의 경험을 제공하도록 반응형으로 디자인되었습니다.

## 3. 프로젝트 구조

```
/
├── index.html              # 메인 랜딩 페이지 (v2 디자인 적용)
├── about.html              # 사이트 소개 페이지 (신규)
├── privacy.html            # 개인정보 처리방침 (v2 디자인 적용)
├── terms.html              # 이용약관 (v2 디자인 적용)
├── 404.html                # 404 페이지 (v2 디자인 적용)
│
├── header.html             # 헤더 플레이스홀더 (v2 디자인 적용)
├── footer.html             # 푸터 플레이스홀더 (v2 디자인 적용)
│
├── style.css               # 기본 스타일 및 폰트 임포트
├── scripts/main.js         # 헤더/푸터 로딩 등 공통 스크립트
├── language.js             # 다국어 지원 스크립트
│
├── lotto/
│   └── index.html          # 로또 번호 생성기 (v2 디자인 적용)
│
├── TextCount/
│   └── index.html          # 글자 수 세기 (v2 디자인 적용)
│
├── eat/
│   └── index.html          # 오늘 뭐 먹지? (v2 디자인 적용)
│
├── Rock-paper-scissors/
│   └── index.html          # 가위바위보 게임 (v2 디자인 적용)
│
├── ladder/
│   └── index.html          # 사다리 게임 (v2 디자인 적용)
│
└── carrot-dodger/
    └── index.html          # 당근 피하기 게임 (v2 디자인 적용)
```

## 4. 기능 명세

- **UI/UX 전면 개편:** 모든 페이지의 UI가 새로운 디자인 시스템(v2)에 맞게 재구성되었습니다.
- **헤더:** '도구', '게임', '소개' 등 명확한 카테고리를 가진 드롭다운 메뉴로 변경하여 사용성을 개선했습니다.
- **메인 페이지:** 사이트의 목적을 명확히 전달하는 Hero 섹션을 추가하고, 각 도구를 소개하는 카드를 현대적인 디자인으로 변경했습니다.
- **도구 및 게임 페이지:** 각 페이지의 콘텐츠를 새로운 카드 레이아웃 안에 배치하고, 입력 필드, 버튼 등의 스타일을 통일하여 일관된 사용자 경험을 제공합니다.
- **정적 페이지:** `개인정보 처리방침`, `이용약관`, `404`, `소개` 페이지 또한 통일된 레이아웃과 스타일을 적용받습니다.

*(기존 각 도구의 핵심 기능은 동일하게 유지됩니다.)*
